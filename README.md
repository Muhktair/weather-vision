A modern, visually rich weather dashboard frontend that presents real-time-style weather data through interactive charts, forecasts, and clean UI components.
This project focuses entirely on the frontend experience, layout, and data visualization.

✨ Features
Modern dark-themed dashboard UI
Live digital clock display
City/location selector
Current weather overview
Hourly temperature forecast (line chart)
Multi-day forecast (4 / 15 / 30 days)
Monthly rainfall visualization (bar chart)
World weather map preview
Temperature unit toggle (°C / °F)
Smooth animations and transitions
Fully responsive layout
